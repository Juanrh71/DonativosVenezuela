# Donativos Venezuela

Aplicación web para reportar necesidades de insumos en zonas afectadas de Venezuela.

## Requisitos
- Node.js 18+

## Ejecutar

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

La interfaz estará disponible en http://localhost:5173 y hablará con el backend en http://localhost:3001.
